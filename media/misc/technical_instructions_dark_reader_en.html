A few preparations before starting

Please turn off browser extensions or add-ons that introduce a dark mode or other website appearance alterations, such as Dark Reader.

Furthermore, please ensure that the experiment takes place on a screen that is not too large; 14" screen (roughly notebook sized) or smaller in normal viewing distance is ideal. If you have only a larger screen available, you may increase the viewing distance such that the screen occupies roughly the same width as it would be a 14" laptop screen in front of you.

If you are using a very small screen (e.g. built in Smartphones) we recommend using landscape mode.
